## 문제

---

<p>두 개의 문자열 <code>str1</code>, <code>str2</code>가 공백으로 구분되어 입력으로 주어집니다.<br>
입출력 예와 같이 <code>str1</code>과 <code>str2</code>을 이어서 출력하는 코드를 작성해 보세요.</p>

```plaintext
- 1 ≤ str1, str2의 길이 ≤ 10
```

## 풀이 코드

---

### 문자열 붙여서 출력하기.cpp

```cpp
#include <iostream>

using namespace std;

int main() {
    string str1, str2, result;
    cin >> str1 >> str2;

    result = str1 + str2;
    cout << result;
}
```
