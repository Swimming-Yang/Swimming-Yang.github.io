두번째 글이자 처음 마크다운을 직접 작성하는 글 입니다.
티스토리로 글을 작성하고 있었는데 디자인의 한계와, 뭔가 불편한 것들이 많아 깃허브 페이지로 옮겨보려고 합니다.

이제 막 2달차로 진입한 C# 방산 개발자로서 아직은 회사 일의 모든것이 어렵고, 낯선 느낌입니다.

MVVM이 뭐며, WPF 같이 처음 들어본 툴을 사용해보기도 하면서 개발자로서 한걸음 앞으로 간 느낌입니다.

```csharp

public class Program
{
    public static void Main()
    {
        Console.WriteLine("꾸준히 현재에 안주하지 않고, 개발이며, 체력이며 유지하며 발전해 나가보려고 합니다.");
    }
}
```

아직은 마크다운으로 무엇을 작성한다는게 어색하고, 블로그 디자인도 AI로 만들어 어색하지만, 조금씩 바꿔보려고 합니다.

언젠가는 여러 개발자, 개발 지망생 분들이 정보를 얻어가는 공간이 되었으면 좋겠습니다.
