실제 포트폴리오에서 사용하는 방법

1. 프로필 사진 교체

현재 index.html의 56번째 줄:

<div class="profile-placeholder">S</div>

이 부분을 다음과 같이 변경:

<img src="images/profile/profile-main.jpg" alt="양수영" class="profile-image">

2. About 섹션 이미지 교체

현재 index.html의 98번째 줄:

<div class="about-placeholder">💼</div>

이 부분을 다음과 같이 변경:

<img src="images/profile/about-photo.jpg" alt="양수영 소개" class="about-image">

3. 프로젝트 이미지 교체

각 프로젝트 카드의 placeholder를 실제 이미지로 교체

4. CSS에서 배경 이미지 사용

.hero {
    background: url('images/background/hero-bg.jpg') center/cover;
}
